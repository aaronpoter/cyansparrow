# Cyan Sparrow 技术支持 / Support

**Cyan Sparrow** 是一款加密货币投资组合和理财记录管理工具。  
Cyan Sparrow is a cryptocurrency portfolio and finance tracker app.

如果在使用过程中遇到问题或有疑问，请参考以下内容。  
If you encounter any issues or have questions while using the app, please refer to the following.

---

## 1. 下载与安装 / Download & Installation

- iOS 用户请在 App Store 下载最新版应用。  
- 确保您的设备系统版本支持 iOS 17 或以上。  
- 如安装遇到问题，请尝试重启设备或更新系统。  

iOS users can download the latest version from the App Store.  
Make sure your device runs iOS 17 or later.  
If you encounter installation issues, try restarting your device or updating iOS.

---

## 2. 用户注册与登录 / Registration & Login

- Cyan Sparrow 使用 **Supabase** 提供安全的账户注册和登录。  
- 注册时需要邮箱和密码。  
- 登录时请确保网络正常。  

Cyan Sparrow uses **Supabase** for secure account registration and login.  
Email and password are required to register.  
Please ensure a stable network connection when logging in.

---

## 3. 功能说明 / Features

- **资产管理 / Asset Management**：添加、删除或修改您的加密货币资产 / Add, remove, or edit your crypto assets.  
- **投资组合统计 / Portfolio Tracker**：查看各资产占比、总余额和收益 / View asset allocation, total balance, and returns.  
- **交易记录 / Transaction History**：记录每笔买入/卖出交易 / Record every buy/sell transaction.  
- **数据安全 / Data Security**：所有信息都存储在 Supabase，数据加密传输 / All data stored in Supabase with encrypted transmission.  

---

## 4. 常见问题（FAQ） / Frequently Asked Questions

1. **数据丢失怎么办？ / Data lost?**  
   - 请确保账户登录正确，Cyan Sparrow 不在本地保存交易数据 / Make sure you are logged in correctly; data is not stored locally.

2. **账户信息安全吗？ / Is my account information secure?**  
   - 所有数据在 Supabase 后端加密存储，通信使用 HTTPS / All data is encrypted in Supabase backend; communication is via HTTPS.
